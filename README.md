# 🏨 Grand Luxe Hotel Website

A fully responsive, multi-page hotel website built with HTML5, CSS3, and JavaScript.

## 🌐 Live Website
[View on Netlify](https://your-netlify-link.netlify.app)

## 📄 Pages
| Page | File | Purpose |
|------|------|---------|
| Home | index.html | Banner, highlights, featured rooms, CTA |
| About | about.html | Hotel story, values, team |
| Rooms | rooms.html | Room types, pricing, amenities |
| Gallery | gallery.html | Photo gallery with lightbox |
| Booking | booking.html | Reservation form + contact details |

## 🛠️ Technologies Used
- **HTML5** – Semantic page structure
- **CSS3** – Responsive layout, Flexbox, Grid, Media Queries
- **JavaScript** – Mobile menu, form validation, gallery lightbox

## ✅ Features
- Fully responsive (mobile, tablet, desktop)
- Basic SEO (title tags, meta descriptions, alt text, heading hierarchy)
- JavaScript form validation with error messages
- Interactive image gallery with lightbox
- Mobile hamburger navigation menu
- Sticky navbar, CTA sections, footer
